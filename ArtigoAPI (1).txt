Métodos HTTP e Status Codes: A Linguagem da Web

As APIs REST, que são as mais comuns, utilizam HTTP para se comunicar. Os métodos HTTP são como verbos que indicam a ação que você quer realizar. sendo os mais usados:

GET: Para buscar dados. Um exemplo seria buscar um obejeto num site de compras por exempo.

POST: Para enviar dados e criar algo novo. Por exemplo criar um novo usuário com dados.

PUT/PATCH: Para atualizar dados. Como atualizar o e-mail de um usuário .

DELETE: Para remover dados. Quero deletar o produto do site por exemplo.

Além dos métodos existem os Status Codes, que são "mensagens de resposta" do servidor. Alguns exemplos bem simples são:

200 OK: Tudo certo

201 Created: Foi criado com sucesso.

404 Not Found: O que você tentou acessar não foi encontrado.

500 Internal Server Error: Erro no servidor.
